# COVID-19-Testing-App
An Android based COVID-19 tracking app which have these informations :-

-COVID 19 Symptoms 
-COVID 19 Precautions 
-COVID 19 Live Patient Counter
-COVID 19 Soft Testing
-COVID 19 Detector

Detail Functionalities of Domain

1. It shows the real time COVID Patients.
2. The user can interact with application in three different languages English , Hindi and Gujarati.
3. Shows the list of COVID Help Centers near by User.
4. Its gives brief information about COVID Symptoms and COVID Precautions.
5. For Quarantine users it have list of Quarantine activities for enjoying the quarantine time period.
6. It also generate the notification at the interval of 2 hours for Sanitization.
7. The user can conducts the COVID Soft Test at home.
8. The conducted test can get response in three language.
